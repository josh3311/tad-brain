# Skill: debug

## Purpose
Debug Python code and explain what went wrong and why.

## Instructions
1. Read the error message carefully
2. Identify the exact line and reason
3. Fix the code
4. Explain why it broke in simple terms Joshua can learn from
5. Suggest how to avoid same mistake next time

## Tools needed
- file_read
- file_write

## Output format
Explain error in plain English first. Show fixed code. Show what changed and why.

## Success criteria
Joshua understands what went wrong, not just that it got fixed.
